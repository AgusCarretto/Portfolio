Proyecto Integrador - Gestion interna a empresa de arquitectos

(A diferencia de los demas proyectos, este fue realizado para un cliente real, de esta forma el desarrollo maneja datos sensibles del mismo, por consecuencia el repositorio no esta disponible.)
El desarrollo de este proyecto se basa en los problemas que tuvo que afrontar el cliente Bodega & Piedrafita Arquitectos en cuanto a procesos de gestión.
La intención del equipo de desarrollo es agilizar y automatizar estos procesos de gestión mediante un sistema de tres módulos en formato de aplicación web que ataque las problemáticas directamente, llevándose registro de los materiales, vinculándose códigos QR a las obras para acceder virtualmente a los planos y automatizando los cálculos de salarios a partir de la información existente de registros de horas.

Para el desarrollo del sistema se utilizarán como tecnologías principales:

  • C# / Asp .NET para el desarrollo principal.
  • SQL Server para el manejo de bases de datos
  • Entity Framework para facilitar el manejo de la capa de persistencia de datos con C#
  • Bootstrap para el diseño gráfico de la página
  • Microsoft Azure para el deploy del mismo

Estas tecnologías fueron seleccionadas para generar una solución de código robusto y seguro (Utilizándose el patrón MVC), para tener un proceso de creación de interfaz de usuario acelerado, y para poder interactuar con los sistemas existentes de la empresa.
